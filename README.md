# Todo List
O todo list é um projeto idealizado para organizar as tarefas do seu dia a dia<br><br>

## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- TypeScript
- React JS
- Styled Components
- Firebase / Realtime Database<br><br>

## Run Locally

### Instalar Dependências
~~~bash
$ npm install
~~~

### Executar Projeto
~~~bash
$ npm run dev
~~~
